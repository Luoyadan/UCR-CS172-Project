Overview
-------

This application uses the Twitter Streaming API to collect geolocated tweets and stores them in text files of 10MB each.

Running the Program
-------
1. Make sure you have tweepy installed (found at https://github.com/tweepy/tweepy) and Python 2.7 or greater
2. If on Linux, run './crawler.sh' and pass in the directory (defaults to /data)
